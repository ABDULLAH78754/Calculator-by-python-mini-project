# Python Calculator 🧮

A simple command-line calculator built using Python.
This calculator performs basic arithmetic operations and allows users to continuously calculate results until exiting the program.

## ✨ Features

* Addition (`+`)
* Subtraction (`-`)
* Multiplication (`*`)
* Division (`/`)
* Modulus (`%`)
* Reset calculator (`r`)
* Show current result (`re`)
* Exit program (`e`)

---

## 🚀 How It Works

1. Enter an initial number.
2. Choose an operator.
3. Enter another number for calculation.
4. Continue performing operations on the current result.
5. Use:

   * `re` → to display current result
   * `r` → to reset calculator
   * `e` → to exit

---

## 🖥️ Example

```bash
*********** Welcome Calculator by Python ***********

Enter Number: 10
Enter Operator ( + - * / %, r to reset, re to result, e to exit): +
Enter Number: 5

Enter Operator ( + - * / %, r to reset, re to result, e to exit): re
Result: 15.0
```
